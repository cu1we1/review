#### 6. 先行断言&正向否定查找

- 先行断言：`x(?=y)` 匹配 x，仅仅当 x 后面跟着 y。
- 正向否定查找：`x(?!y)`匹配 x，仅仅当 x 后面没有跟着 y。

从字面意思就可以看出来，这两个是相反的匹配规则。

看个例子：

```js
// 从'this is vue'中取出is中的i。
const reg = /(i)(?=s)/g;
[..."this is vue".matchAll(reg)];
/**
[
    {
        "i",
        "i",
        groups: undefined,
        index: 2,
        input: "this is vue",
    },
    {
        "i",
        "i",
        groups: undefined,
        index: 5,
        input: "this is vue",
    }
]
**/
```
